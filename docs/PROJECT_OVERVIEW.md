# Project Overview

## What this is

**Kaizer Chiefs × Bitexen — Building the Digital Fan Economy** is a standalone,
single-page proposal website. It presents a strategic vision for building a
digital supporter economy around the Kaizer Chiefs supporter base, powered by
Bitexen's digital asset infrastructure.

The site is a long-form, scrollable narrative derived from the
*Kaizer Chiefs Presentation (v1)* deck. It walks the reader from the macro shift
in sport spectatorship, through the digital fan economy model and its commercial
case, to the proposed partnership and a closing discussion.

## Target audience

- **Primary:** Kaizer Chiefs leadership and commercial decision-makers.
- **Secondary:** Bitexen stakeholders and partnership leads presenting the vision.

The site is access-gated by a lock screen (access code **1970**) so it can be
shared privately during the proposal process.

## Deployment environment

- **Framework:** Next.js (App Router) + TypeScript.
- **Styling:** Tailwind CSS v4 with a custom Kaizer Chiefs / Bitexen theme.
- **UI primitives:** shadcn-style components (Radix under the hood).
- **Hosting:** Vercel (Next.js preset). See `DEPLOYMENT.md`.
- **Repository:** https://github.com/broadbrandsa/bitexen-kc-v2
