# Kaizer Chiefs × Bitexen — Digital Fan Economy Proposal

Standalone, single-page proposal website presenting the vision for building a
digital supporter economy around the Kaizer Chiefs supporter base, powered by
Bitexen's digital asset infrastructure. Access-gated by a lock screen.

## Stack

- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS v4 (CSS-first theme, no `tailwind.config`)
- shadcn/ui primitives (button, card, badge, accordion, tabs, separator, dialog, dropdown-menu)
- ESLint
- pnpm
- `src/` directory layout
- No experimental flags

## Run locally (port 3002)

```bash
pnpm install
pnpm dev
```

Open http://localhost:3002 and enter access code **1970**.

## Build

```bash
pnpm build
pnpm start
```
